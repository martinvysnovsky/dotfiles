# SEO Checklists

## Complete Page SEO Audit Checklist

### Technical SEO
- [ ] **Page loads in <3 seconds** (mobile and desktop)
- [ ] **HTTPS enabled** (valid SSL certificate)
- [ ] **Mobile-friendly** (responsive design)
- [ ] **No broken links** (internal or external)
- [ ] **Canonical tag present** (self-referencing or pointing to preferred version)
- [ ] **Proper status code** (200 for live pages)
- [ ] **No redirect chains** (maximum 1 redirect)
- [ ] **XML sitemap includes page**
- [ ] **Robots.txt doesn't block page**
- [ ] **Core Web Vitals passing** (LCP <2.5s, FID <100ms, CLS <0.1)

### On-Page SEO

#### Title Tag
- [ ] **Present and unique**
- [ ] **50-60 characters** (displays fully in SERPs)
- [ ] **Includes primary keyword** (preferably at beginning)
- [ ] **Compelling and clickworthy**
- [ ] **Matches search intent**
- [ ] **Includes brand name** (at end, if space allows)

#### Meta Description
- [ ] **Present and unique**
- [ ] **150-160 characters** (displays fully in SERPs)
- [ ] **Includes target keyword naturally**
- [ ] **Clear value proposition**
- [ ] **Contains call-to-action**
- [ ] **Compelling and encourages clicks**

#### Heading Structure
- [ ] **Single H1 tag** (contains primary keyword)
- [ ] **H2 tags for major sections** (contain related keywords)
- [ ] **Logical hierarchy** (no skipped levels)
- [ ] **Descriptive headings** (outline content clearly)
- [ ] **Keywords used naturally** (not stuffed)

#### Content Quality
- [ ] **Original content** (not duplicated from other pages)
- [ ] **Adequate length** (1,500+ words for competitive topics)
- [ ] **Keyword in first 100 words**
- [ ] **Primary keyword in at least one H2**
- [ ] **Natural keyword density** (1-2%, not stuffed)
- [ ] **LSI keywords included** (related terms and synonyms)
- [ ] **Answers search intent** (matches what users are looking for)
- [ ] **Easy to read** (short paragraphs, bullet points, formatting)
- [ ] **Up-to-date information** (updated within 12 months)
- [ ] **Clear structure** (introduction, body, conclusion)

#### Internal Linking
- [ ] **3-5 relevant internal links** (contextual, within content)
- [ ] **Descriptive anchor text** (keyword-rich, not "click here")
- [ ] **Links to related content**
- [ ] **Receives inbound links** (not an orphan page)
- [ ] **Links to important pages** (products, services, conversions)

#### Images
- [ ] **All images have alt text** (descriptive, includes keywords when relevant)
- [ ] **Images compressed** (<200KB each)
- [ ] **Modern format used** (WebP preferred)
- [ ] **Descriptive file names** (keyword-descriptive.jpg)
- [ ] **Appropriate dimensions** (not oversized then scaled down)
- [ ] **Lazy loading enabled** (for below-fold images)
- [ ] **Responsive images** (srcset for multiple sizes)

#### URL Structure
- [ ] **Short and descriptive** (3-5 words)
- [ ] **Includes target keyword**
- [ ] **Uses hyphens** (not underscores)
- [ ] **All lowercase**
- [ ] **No unnecessary parameters** (clean URLs)
- [ ] **Logical hierarchy** (reflects site structure)

### Schema Markup
- [ ] **Schema implemented** (appropriate type for content)
- [ ] **Valid JSON-LD** (no syntax errors)
- [ ] **Tested in Rich Results Test**
- [ ] **All required properties included**
- [ ] **Enhances search appearance** (rich snippets potential)

**Common schema types:**
- [ ] Article (blog posts)
- [ ] Product (product pages)
- [ ] FAQ (FAQ sections)
- [ ] Breadcrumb (navigation)
- [ ] Organization (homepage)
- [ ] LocalBusiness (local companies)

### User Experience
- [ ] **Clear call-to-action** (above the fold)
- [ ] **Easy navigation** (clear menu, breadcrumbs)
- [ ] **No intrusive interstitials** (pop-ups that block content)
- [ ] **Readable font size** (16px minimum)
- [ ] **Sufficient color contrast** (WCAG AA compliance)
- [ ] **Touch targets sized appropriately** (48x48px minimum)
- [ ] **No horizontal scrolling** (on mobile)

### Conversions
- [ ] **Clear conversion path** (easy to take action)
- [ ] **Trust signals present** (testimonials, reviews, badges)
- [ ] **Contact information visible**
- [ ] **Social proof included** (customer count, ratings, logos)
- [ ] **Minimal distractions** (focused on primary goal)

## Content Optimization Checklist

### Before Writing

#### Research
- [ ] **Keyword research completed** (primary + secondary keywords identified)
- [ ] **Search intent understood** (informational, commercial, transactional)
- [ ] **Competitor analysis done** (top 5 ranking pages reviewed)
- [ ] **Content gaps identified** (topics competitors miss)
- [ ] **Target word count determined** (based on top-ranking content)

#### Planning
- [ ] **Content outline created** (H2/H3 structure planned)
- [ ] **LSI keywords identified** (related terms to include)
- [ ] **Internal linking opportunities mapped** (relevant pages to link to)
- [ ] **Visual assets planned** (images, charts, videos needed)

### During Writing

#### Content Structure
- [ ] **Compelling introduction** (hooks reader, states value)
- [ ] **Logical flow** (ideas progress naturally)
- [ ] **Subheadings every 300-400 words** (improves scannability)
- [ ] **Short paragraphs** (2-4 sentences max)
- [ ] **Bullet points and lists** (breaks up text)
- [ ] **Examples included** (makes concepts concrete)
- [ ] **Clear conclusion** (summarizes key points)
- [ ] **Call-to-action** (tells reader what to do next)

#### Keyword Usage
- [ ] **Primary keyword in H1**
- [ ] **Primary keyword in first 100 words**
- [ ] **Primary keyword in at least one H2**
- [ ] **Primary keyword in URL**
- [ ] **Primary keyword in meta description**
- [ ] **Primary keyword used naturally throughout** (1-2% density)
- [ ] **Secondary keywords in H2/H3 tags**
- [ ] **LSI keywords included** (synonym variations)
- [ ] **Long-tail variations used** (question-based keywords)

#### Readability
- [ ] **Active voice used** (not passive)
- [ ] **Simple language** (8th-10th grade reading level)
- [ ] **Short sentences** (average 15-20 words)
- [ ] **Transitions between sections** (guides reader)
- [ ] **Jargon explained** (or avoided)
- [ ] **Scannable format** (bold, italics for emphasis)

### After Writing

#### Final Review
- [ ] **Spell-check completed**
- [ ] **Grammar check completed**
- [ ] **Fact-check completed** (statistics verified)
- [ ] **Links tested** (all work correctly)
- [ ] **Plagiarism check** (100% original)
- [ ] **Mobile preview** (reads well on phone)

#### SEO Elements
- [ ] **Title tag optimized** (50-60 chars, keyword-rich)
- [ ] **Meta description optimized** (150-160 chars, compelling)
- [ ] **URL slug optimized** (short, includes keyword)
- [ ] **Image alt text added** (all images)
- [ ] **Internal links added** (3-5 contextual links)
- [ ] **External links added** (authoritative sources)
- [ ] **Schema markup added** (if applicable)

#### Visuals
- [ ] **Featured image added** (high quality, relevant)
- [ ] **Supporting images added** (break up text)
- [ ] **Images compressed** (fast loading)
- [ ] **Charts/graphs included** (for data-heavy content)
- [ ] **Video embedded** (if relevant)

## Technical SEO Audit Checklist

### Crawling & Indexing
- [ ] **XML sitemap exists** (and is up-to-date)
- [ ] **Sitemap submitted to Search Console**
- [ ] **Robots.txt configured correctly** (not blocking important pages)
- [ ] **All important pages indexed** (site:example.com in Google)
- [ ] **No duplicate content issues** (canonical tags used)
- [ ] **No soft 404 errors** (pages return correct status codes)
- [ ] **Pagination handled correctly** (rel="next/prev" or view-all page)

### Site Architecture
- [ ] **Logical URL structure** (follows site hierarchy)
- [ ] **Maximum 3 clicks to any page** (from homepage)
- [ ] **Breadcrumb navigation implemented**
- [ ] **Internal linking strategy** (important pages have most links)
- [ ] **No orphan pages** (every page accessible)
- [ ] **Categories organized logically**

### Page Speed
- [ ] **Core Web Vitals passing** (LCP, FID, CLS)
- [ ] **Images optimized** (compressed, WebP format)
- [ ] **CSS minified**
- [ ] **JavaScript minified**
- [ ] **HTML minified**
- [ ] **Browser caching enabled**
- [ ] **CDN implemented** (for global sites)
- [ ] **Server response time <200ms**
- [ ] **Critical CSS inlined**
- [ ] **Non-critical CSS deferred**
- [ ] **JavaScript deferred/async** (non-critical scripts)
- [ ] **Fonts optimized** (preloaded, font-display: swap)

### Mobile Optimization
- [ ] **Responsive design** (adapts to all screen sizes)
- [ ] **Viewport meta tag present**
- [ ] **Touch targets appropriately sized** (48x48px)
- [ ] **Text readable without zooming** (16px minimum)
- [ ] **No horizontal scrolling**
- [ ] **Mobile page speed optimized** (<3s load time)
- [ ] **No mobile-specific errors** (in Search Console)

### Security
- [ ] **HTTPS enabled** (valid SSL certificate)
- [ ] **All resources load via HTTPS** (no mixed content)
- [ ] **HTTP redirects to HTTPS** (301 permanent)
- [ ] **HSTS header implemented**
- [ ] **Security headers configured** (CSP, X-Frame-Options)

### International SEO (if applicable)
- [ ] **Hreflang tags implemented** (for multi-language sites)
- [ ] **Correct hreflang format** (language-country codes)
- [ ] **Self-referencing hreflang** (each page includes itself)
- [ ] **X-default specified** (fallback version)

## Ecommerce SEO Checklist

### Product Pages
- [ ] **Unique product descriptions** (not manufacturer copy)
- [ ] **300-500 words minimum** (for competitive products)
- [ ] **Keywords in title** (product name + key attribute)
- [ ] **Compelling meta description** (includes price, benefits, CTA)
- [ ] **High-quality images** (multiple angles, zoom capability)
- [ ] **Image alt text optimized** (product name + attributes)
- [ ] **Product schema markup** (price, availability, rating)
- [ ] **User reviews visible** (boosts conversion and SEO)
- [ ] **Related products linked** (internal linking)
- [ ] **Breadcrumb navigation** (shows category hierarchy)
- [ ] **Clear call-to-action** (Add to Cart button prominent)
- [ ] **Trust signals** (return policy, guarantees, security badges)

### Category Pages
- [ ] **Unique category descriptions** (300-500 words)
- [ ] **Description above products** (SEO text visible to crawlers)
- [ ] **Keyword-optimized title tag**
- [ ] **Faceted navigation handled** (canonicals prevent duplicate content)
- [ ] **Subcategories linked** (logical hierarchy)
- [ ] **Products per page optimized** (balance UX and load time)
- [ ] **Pagination implemented correctly** (rel="next/prev" or load more)

### Site Structure
- [ ] **Flat architecture** (products reachable in 3 clicks)
- [ ] **Logical categories** (clear, intuitive organization)
- [ ] **Filter/sort doesn't create duplicates** (use canonicals or noindex)
- [ ] **Search results not indexed** (noindex on search result pages)
- [ ] **Out-of-stock products handled** (keep indexed, mark availability)

### Technical
- [ ] **Fast page load** (<2s for product pages)
- [ ] **Mobile-optimized** (easy to browse and purchase on phone)
- [ ] **Structured data** (Product, Offer, AggregateRating)
- [ ] **HTTPS for entire site** (especially checkout)
- [ ] **No duplicate content** (from multiple URLs for same product)

## Quick Win SEO Checklist

### 30-Minute SEO Improvements

- [ ] **Add missing alt text** (images without descriptions)
- [ ] **Fix broken links** (404 errors, redirects)
- [ ] **Add internal links** (to important pages from high-traffic pages)
- [ ] **Optimize title tags** (pages with generic titles)
- [ ] **Add meta descriptions** (pages missing descriptions)
- [ ] **Compress large images** (>500KB files)
- [ ] **Enable browser caching**
- [ ] **Submit updated sitemap** (if new pages added)
- [ ] **Fix mobile usability issues** (from Search Console)
- [ ] **Add schema markup** (to top-performing pages)

### Monthly SEO Maintenance

- [ ] **Check Search Console errors** (coverage, mobile, enhancements)
- [ ] **Monitor rankings** (top keywords and pages)
- [ ] **Update old content** (refresh posts >12 months old)
- [ ] **Add new content** (1-2 blog posts minimum)
- [ ] **Build backlinks** (outreach, guest posts)
- [ ] **Review analytics** (traffic trends, top pages, conversions)
- [ ] **Test page speed** (monitor Core Web Vitals)
- [ ] **Check for broken links** (use Screaming Frog or similar)
- [ ] **Review competitor activity** (new content, rankings)
- [ ] **Update local listings** (hours, photos, posts)
